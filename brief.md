# Бриф: Тимченко.про — сайт компании полного цикла (инженерные системы)

Дата: 2026-07-18. Источник: интервью Этапа 0 (ответы заказчика по всем 11 блокам).

## Дизайн-ридинг

Reading this as: многостраничный презентационный сайт + лид-каталог для владельцев частных домов Краснодарского края, с премиальным «студийным» языком архитектурных бюро: тёмный глубокий графит, крупная типографика, много воздуха, строгая футуристичность. Стек Next.js + TypeScript + Tailwind + shadcn/ui, сдержанный интерактив (Motion/GSAP точечно).

Дизайн-дайлы (baseline для Этапа 2): `DESIGN_VARIANCE: 7-8`, `MOTION_INTENSITY: 6`, `VISUAL_DENSITY: 3`.

## 1. Бизнес и позиционирование

- Бренд: **Тимченко.про**, юр. форма ИП, 10 лет на рынке.
- Суть оффера: «Вся инженерия вашего дома в одних руках: от проектирования до пуско-наладки. Один подрядчик. Один номер телефона. Никакой головной боли.»
- Отличия: своё производство узлов; сроки; гарантия; инженер с 10+ лет опыта; под ключ без субподряда.

## 2. Направления и услуги

- Направления: водоснабжение, канализация, отопление, котельные, электрика + электрощиты, вентиляция, кондиционирование.
- Полный цикл, 7 шагов: выезд/замер → проект → поставка оборудования → монтаж → пуско-наладка → сдача → сервис.
- Гарантия 2 года (минимум по закону); сервисное/гарантийное обслуживание, шеф-монтаж, выезд по гарантии.

## 3. Витрина / каталог

- Модель на старте: **лид-каталог** (изделие → заявка/консультация/расчёт); полноценная витрина позже.
- Категории: готовые узлы, котельные, автоматика для котельных.
- Товарные данные готовятся отдельно; строим на плейсхолдерах.

## 4. Аудитория и география

- Заказчики: владельцы частных домов (основной фокус) + прорабы.
- Клиенту важны качество и гарантии. Главная бьёт точно в частного клиента.
- География: только Краснодарский край: Краснодар, Сочи, Геленджик, Абрау-Дюрсо.

## 5. Бренд и айдентика

- Старый стиль (navy/gold + blueprint-сетка) в архив, не используем.
- Направление: тёмный глубокий графит, премиум «студийный» уровень (архитектурные/дизайн-бюро): дорого, футуристично, строго, минимально, много воздуха, крупная типографика.
- Ассеты: много фото объектов, есть портфолио по объектам.

## 6. Референсы

- makedes.ru, sima-marmor.at, apple.com, tesla.com/solarpanels.
- Общая тема референсов: тёмный графит.

## 7. Контент и доказательства

- Портфолио: ~8 объектов.
- Бренды-партнёры: Stiebel Eltron, FAR, Гейзер, Stout, Stahlmann, Baxi, Viega, Нептун, Rehau (логотипы брать с Lunda.ru).
- Тексты пишем с нуля, потом правим.

## 8. Функциональность (дорожная карта)

- **Фаза 1 (оболочка):** кнопки-слоты под будущие квизы по направлениям + рабочая заявка на просчёт. Поля заявки: что просчитать / номер телефона / удобное время связи. Мессенджеры: Telegram, WhatsApp, MAX, Instagram.
- **Позже (квизы):** подбор готового узла ввода по параметрам квартиры; расчёт котельной под дом; «какие инженерные системы вам нужны»; просчёт электрики и умного дома. Формат: один квиз с ветками ИЛИ кнопка на каждом направлении — решим отдельно.
- **Позже:** калькуляторы (место определим отдельно). Существующий калькулятор узла ввода (`/calc`) — судьба уточняется.

## 9. Лиды и конверсия

- Целевое действие: **заявка на просчёт**.
- Лиды на старте: только в Telegram.
- Лид-магниты: опционально, позже.

## 10. Техника и организация

- Хостинг: временно GitHub Pages (статика, значит Next.js static export и отправка форм через сторонний сервис/бот) → потом перенос на полноценный хост.
- Домен: в процессе покупки (вероятно, тимченко.про). Текущего сайта нет — делаем с нуля.
- Язык: русский.

## 11. SEO / продвижение / интеграции

- SEO и продвижение — вне скоупа (заказчик сам).
- Аналитика / чат / карты: по желанию, позже.
