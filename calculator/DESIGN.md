# Timchenko.pro Calculator — дизайн-система

## Бренд

| Токен | Значение | Использование |
|---|---|---|
| `navy` | `#1F3864` | Основной текст, secondary-кнопки, тёмная summary-панель |
| `navy-deep` | `#16294A` | Текст на золотом, глубокий фон |
| `gold` | `#BF9000` | Primary-кнопки, акценты, выбранные состояния |
| `gold-bright` | `#D9A400` | Hover primary |
| `paper` | `#ECF0F5` | Фон страницы |
| `paper-2` | `#F6F8FB` | Фон карточек |
| `success` | `#1E7D45` | WhatsApp CTA, «включено», завершённые шаги |

Все цвета компонентов идут через shadcn-переменные (`--primary`, `--secondary`, …) в `globals.css` — компоненты не знают hex-значений.

## Типографика

- **Space Grotesk** — заголовки и UI-текст (`font-sans`)
- **IBM Plex Mono** — цифры, цены, метки шагов, eyebrow (`font-mono`)
- База 16px, line-height 1.5+. Цены — только mono (табличные цифры).

## Стиль: blueprint

- Чертёжная сетка 32px (`.blueprint-bg` / `.blueprint-bg-dark`) на фоне страницы и summary-панели
- Радиусы почти квадратные (`--radius: 2px`) — «чертёжная» жёсткость
- Mono-эйрбоу капсом с трекингом `.1em+` как подписи листов чертежа («ШАГ 1 ИЗ 3», «СХ-01»)

## Паттерн: конфигуратор со sticky-итогом

- Слева — 3 шага (объект → комплектация → опции), все видны сразу, без wizard-блокировки
- Справа — sticky-панель: состав по позициям, вилка цены, CTA (WhatsApp + телефон)
- На мобильном итог дублируется в закреплённой нижней панели
- Цена — всегда вилка ±, с явной оговоркой «черновая, смету составит инженер»

## Обратная связь на каждом шаге

- Прогресс-степпер сверху: выполненные шаги — зелёная галочка, текущий — золотая подсветка (`aria-current="step"`)
- Галочка в заголовке каждой карточки, когда шаг заполнен
- Цена пульсирует 320ms при изменении (`motion-safe:` — уважает `prefers-reduced-motion`)
- Итоговая вилка в `aria-live="polite"` — скринридер озвучивает пересчёт
- Подсказка «для вашей площади обычно берут …» — кликабельная, применяет рекомендацию

## Доступность

- Все контролы — нативные label + Radix-примитивы (клавиатура/фокус из коробки)
- Радиогруппы в `fieldset`/`legend`, слайдер с `aria-label`, кнопки +/- с `aria-label`
- Touch-таргеты ≥ 44px, focus-visible кольца `--ring` (gold) везде
- Контраст: текст — navy на светлом (≥ 10:1); gold используется только для акцентов и заливок, не для мелкого текста (для текста — затемнённый `hsl(45 100% 30%)`)
- `prefers-reduced-motion` глобально отключает анимации

## Ограничения

- Без бэкенда: CTA ведут в WhatsApp (`wa.me`) с собранным текстом расчёта и на `tel:`
- Статический экспорт Next.js, `basePath /plapforma/calc` — деплой на GitHub Pages рядом с лендингом
